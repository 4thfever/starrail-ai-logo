# 根据参考文字生成 Logo 文字

## 需要作为参考输入的图片

- 原始完整 logo：`assets/original.png`，用于理解最终合成关系和整体品牌识别度。
- 原始文字图：`assets/text.png`，作为字形、排版、比例、横线和文字层级的主要参考。
- 原始背景图：`assets/bg.png`，用于判断文字应如何与原始背景结构配合，避免生成无法合成的光效和边缘。

## 需要输入的参考文字

将以下占位内容替换为具体参考文字：

```text
[REFERENCE_TEXT]
```

## Prompt

请基于参考文字，为《崩坏：星穹铁道》logo 生成一张新的透明背景文字层图像。

必须同时参考上传的原始完整 logo、原始文字图和原始背景图。原始文字图决定字形、排版、比例、笔画轮廓、主副标题层级和底部横线结构；参考文字只提供主题风格。请不要改变文字内容，必须仍然是“崩坏 星穹铁道”的 logo 文字，并保持原始文字的整体识别度。
请保持宽幅游戏 logo 文字构图，而不是竖版海报构图或方形头像构图。

先从参考文字中提取视觉语言，包括主色、辅助色、情绪、材质、光效、代表性符号、动态元素和形状语言。然后把这些特征转译到文字层中，例如文字填充、渐变、描边、内纹理、外发光、笔画边缘、局部符号替换、横线光效和少量主题装饰。

文字的改动要有鲜明主题特色，但不能把原始《崩坏：星穹铁道》logo 的字形辨认度改没。理想效果是：第一眼仍然读得出并认得出原 logo，第二眼能看出它被参考文字描述的主题风格化了。

请遵守以下约束：

- 输出必须是透明背景，只包含文字、文字描边、文字发光、底部横线和必要的局部装饰。
- 不要生成实色背景、渐变背景、场景背景、贴图底板或大面积阴影底图。
- 必须保留“崩坏”和“星穹铁道”的文字内容，不要改字、漏字、错字、添加新文字或生成乱码。
- 保留原始字形的主要轮廓、粗细关系、倾斜感、字距、主副标题层级和横线位置。
- 可以改变文字颜色、金属/能量/材质纹理、描边、内发光、外发光和局部装饰。
- 可以在原本的星形、点状装饰、笔画交叉处或横线附近加入一个最有代表性的主题符号，但要克制，不要让符号替代文字主体。
- 大字可以承载较复杂的材质和纹理，小字“崩坏”必须更清晰简洁，优先保证可读性。
- 参考 `assets/bg.png` 的构图，文字的边缘光和投影方向应能与背景自然合成。
- 发光和粒子只能围绕文字轮廓少量出现，不能扩散成背景。
- 不要生成角色、场景、物体主体或独立插画。

输出要求：

- 透明背景 PNG 风格的文字层。
- 只包含《崩坏：星穹铁道》logo 文字和必要装饰。
- 字形清晰、边缘干净、适合叠加到背景层。
- 主题特色明确，但原始 logo 文字辨认度必须优先。
